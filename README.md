# sonic-screen-library

sonic-screen-library is the screen management library for SonicDE Workspaces. Its primary consumer is the SonicDE screen management application sonic-screen.

## End user

Since this is a development library end users should instead look for support directly for the apps using this library. Please contact the support channels of your Linux distribution first. In case you find a bug in KScreen or if the bug is traced back to sonic-screen-library, you can report it at the sonic-screen-library [bug tracker](https://github.com/Sonic-DE/sonic-screen-library/issues) (first look for duplicates).

## Contributing

We appreciate your interest in contributing! To report a bug, please use the Plasma Workspace bug tracker at [Issues · Sonic-DE/plasma-workspace-sonic](https://github.com/Sonic-DE/plasma-workspace-sonic/issues).

## Build instructions

sonic-system-info uses CMake like most SonicDE projects, so you can build it like this:

```bash
mkdir build
cd build
cmake ..
make
```

or with ninja:

```bash
mkdir build
cd build
cmake -G Ninja ..
ninja
```
